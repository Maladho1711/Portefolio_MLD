# Ajouter votre photo de profil

Pour ajouter votre photo sur la page d'accueil :

1. **Placez votre photo dans ce dossier** (`public/images/`)
2. **Nommez-la** `profile.jpg` ou `profile.png`
3. La photo s'affichera automatiquement sur la page d'accueil !

**Recommandations pour la photo :**
- Format : JPG ou PNG
- Taille recommandée : 800x800px minimum (carré)
- Fond : Photo professionnelle avec fond neutre ou transparent
- Qualité : Haute résolution pour un rendu optimal
- Poids : Optimisez l'image (moins de 500KB de préférence)

**Note :** Si vous ne placez pas de photo, un avatar stylisé avec vos initiales s'affichera automatiquement.
